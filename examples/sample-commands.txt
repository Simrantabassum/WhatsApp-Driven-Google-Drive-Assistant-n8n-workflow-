## Testing Checklist

### Basic Functionality
- [ ] HELP command returns help text
- [ ] LIST command shows files in folder
- [ ] MOVE command moves files successfully
- [ ] DELETE command requests confirmation
- [ ] CONFIRM DELETE executes deletion

### AI Features
- [ ] SUMMARY command processes files
- [ ] AI generates meaningful summaries
- [ ] Handles different file types (PDF, DOCX, TXT)
- [ ] Respects file size limits

### Error Handling
- [ ] Invalid commands return helpful error messages
- [ ] Non-existent files/folders handled gracefully
- [ ] Missing arguments provide usage instructions
- [ ] API errors are communicated clearly

### Security Features
- [ ] Delete operations require confirmation
- [ ] Sensitive information not exposed in responses
- [ ] Input validation prevents injection attacks

### Performance
- [ ] Responses received within reasonable time
- [ ] Large file lists handled efficiently
- [ ] AI summarization completes in acceptable time
